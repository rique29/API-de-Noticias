# Notícias API

Esta é uma API RESTful simples para gerenciar notícias.

## Endpoints

### Listar Notícias
GET /news

### Obter Notícia Específica
GET /news/{id}

### Criar Nova Notícia
POST /news

### Atualizar Notícia
PUT /news/{id}

### Deletar Notícia
DELETE /news/{id}
